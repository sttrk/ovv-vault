# カレー

## Description

調理済みで食べられる状態のカレー。具材が適切に調理され、スパイスが効いた料理。

## Scope

### In Scope

- 具材の準備（玉ねぎ、にんじん、じゃがいも、肉）
- スパイスやルーの使用
- 調理プロセス全般（炒める、煮込むなど）
- 完成したカレーの提供

### Out of Scope

- ご飯の準備
- サラダや副菜の準備
- カレー以外の料理

## Completion Criteria

- 具材が適切に調理されていること
- スパイスやルーが均一に混ざり、味が整っていること
- 食べられる状態で提供可能であること

## State Transition

| 項目 | 値 |
|------|-----|
| 開始状態 | 具材や調理器具が未準備の状態 |
| 終了状態 | 完成したカレーが食べられる状態で提供されている |

## Assumptions

- 必要な具材と調理器具が揃っている
- 調理を行うためのキッチンが利用可能である
- 調理者が基本的な調理スキルを持っている
